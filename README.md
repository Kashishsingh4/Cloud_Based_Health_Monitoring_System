# Cloud-Based Health Monitoring System

A cloud-based IoT health monitoring system that tracks patient vitals such as **heart rate**, **body temperature**, and **oxygen saturation (SpO₂)** in real time. Data is securely stored on the **cloud** and accessible through a **web or mobile dashboard**, enabling remote monitoring, instant alerts, and affordable, scalable healthcare management.

## Features
- Real-time monitoring of patient vitals  
- Cloud storage and secure data access  
- Remote dashboard for doctors and caregivers  
- Instant alerts for abnormal readings  
- Scalable and cost-effective design
